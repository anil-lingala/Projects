# Machine Learning Projects

This directory contains my machine learning projects, organized by topic.  
Each folder focuses on a specific ML concept and includes code, experiments, and documentation.

## Projects

### Classification – Ensemble Methods
Explores ensemble learning techniques for classification tasks.
- Methods such as bagging, boosting, and voting classifiers
- Focus on improving accuracy and robustness

### KMeans & Expectation-Maximization (EM)
Unsupervised learning projects focused on clustering.
- K-Means clustering
- Expectation-Maximization algorithm
- Analysis and comparison of clustering behavior

### MLP & CNN Networks
Neural network–based models for classification tasks.
- Multi-Layer Perceptrons (MLP)
- Convolutional Neural Networks (CNN)
- Experiments with different architectures and hyperparameters

### Text Classification
Projects related to natural language processing and text classification.
- Feature extraction from text
- Traditional ML and/or neural approaches
- Evaluation of classification performance

## Technologies Used
- Python
- NumPy, Pandas
- Scikit-learn
- TensorFlow / PyTorch (where applicable)
- Jupyter Notebooks
